# NAME

gui_zoom_to - gui zoom to

# SYNOPSIS

gui::zoom_to 
    x0 y0 x1 y1


# DESCRIPTION

To zoom in our out to a specific region:

# OPTIONS

`x0, y0, x1, y1`:  first and second corner of the layout area in microns

# ARGUMENTS

This command has no arguments.
