# NAME

create_logic_port - create logic port

# SYNOPSIS

create_logic_port
    [-direction direction]
    port_name


# DESCRIPTION

This command creates logic port. Direction must be specified from:
`in`, `out`, `inout`.

# OPTIONS

`-direction`:  Direction of the port (`in`, `out`, `inout`).

`port_name`:  Port name.

# ARGUMENTS

This command has no arguments.
