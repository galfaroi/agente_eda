# NAME

remove_io_bump - remove io bump

# SYNOPSIS

remove_io_bump 
    instance_name


# DESCRIPTION

This command removes a single bump instance.

# OPTIONS

`instance_name`:  Name of the bump.

# ARGUMENTS

This command has no arguments.
