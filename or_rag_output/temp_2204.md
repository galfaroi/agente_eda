# NAME

gui_pause - gui pause

# SYNOPSIS

gui::pause 
    [timeout]


# DESCRIPTION

Pause the execution of the script:

# OPTIONS

`timeout`:  is specified in milliseconds, if it is not provided the pause will last until the user presses the Continue button

# ARGUMENTS

This command has no arguments.
