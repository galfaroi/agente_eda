# NAME

remove_io_bump_array - remove io bump array

# SYNOPSIS

remove_io_bump_array 
    -bump master


# DESCRIPTION

This command removes the entire bump array.

Example usage:

```
remove_io_bump_array -bump BUMP
```

# OPTIONS

`-bump`:  Name of the bump master.

# ARGUMENTS

This command has no arguments.
