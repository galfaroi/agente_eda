# NAME

report_design_area - report design area

# SYNOPSIS

report_design_area


# DESCRIPTION

The `report_design_area` command reports the area of the design's components
and the utilization.

# OPTIONS

This command has no switches.

# ARGUMENTS

This command has no arguments.
