# NAME

unset_dont_touch - unset dont touch

# SYNOPSIS

unset_dont_touch instances_nets


# DESCRIPTION

The `unset_dont_touch` command reverse the `set_dont_touch` command.

# OPTIONS

This command has no switches.

# ARGUMENTS

This command has no arguments.
