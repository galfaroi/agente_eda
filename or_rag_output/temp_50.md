# NAME

report_dft_config - report dft config

# SYNOPSIS

report_dft_config


# DESCRIPTION

Prints the current DFT configuration to be used by `preview_dft` and
`insert_dft`.

# OPTIONS

This command has no switches.

# ARGUMENTS

This command has no arguments.
