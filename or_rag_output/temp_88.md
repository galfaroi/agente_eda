# NAME

focus_net - focus net

# SYNOPSIS

focus_net 
    net
    [-remove]
    [-clear]


# DESCRIPTION

This command limits the drawing to specified net.

# OPTIONS

`pin`:  name of the net.

`-remove`:  (optional) removes the net from from the focus.

`-clear`:  (optional) clears all nets from focus.

# ARGUMENTS

This command has no arguments.
