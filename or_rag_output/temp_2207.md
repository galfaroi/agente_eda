# NAME

gui_remove_toolbar_button - gui remove toolbar button

# SYNOPSIS

gui::remove_toolbar_button 
    name


# DESCRIPTION

To remove toolbar button:

# OPTIONS

`-name`:  The name of the button, used when deleting the button.

# ARGUMENTS

This command has no arguments.
