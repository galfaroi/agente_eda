# NAME

gui_hide_widget - gui hide widget

# SYNOPSIS

gui::hide_widget 
    name


# DESCRIPTION

To close a specific layout widget:

# OPTIONS

`name`:  of the widget. For example, the display controls would be "Display Control".

# ARGUMENTS

This command has no arguments.
