# NAME

gui_show_widget - gui show widget

# SYNOPSIS

gui::show_widget 
       name


# DESCRIPTION

To open a specific layout widget:

# OPTIONS

`name`:  of the widget. For example, the display controls would be "Display Control".

# ARGUMENTS

This command has no arguments.
