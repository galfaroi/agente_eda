# NAME

unset_dont_use - unset dont use

# SYNOPSIS

unset_dont_use lib_cells


# DESCRIPTION

The `unset_dont_use` command reverses the `set_dont_use` command.

# OPTIONS

This command has no switches.

# ARGUMENTS

This command has no arguments.
