# NAME

gui_remove_menu_item - gui remove menu item

# SYNOPSIS

gui::remove_menu_item 
    name


# DESCRIPTION

To remove menu item:

# OPTIONS

`-name`:  name of the item, used when deleting the item

# ARGUMENTS

This command has no arguments.
