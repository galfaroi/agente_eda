# NAME

bench_read_spef - bench read spef

# SYNOPSIS

bench_read_spef
    [filename]                   


# DESCRIPTION

The `bench_read_spef` command reads a `<filename>.spef` file and stores the
parasitics into the database.

# OPTIONS

`filename`:  Path to the input `.spef` file.

# ARGUMENTS

This command has no arguments.
